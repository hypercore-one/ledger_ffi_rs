# Ledger FFI library

FFI communication library between Rust and Ledger Nano S/X devices.

## Build from source

```bash
git clone https://github.com/zenon-network/ledger_ffi_rs.git
cd ledger_ffi_rs
cargo build
```

The generated library can be found in the `./target/` folder.
